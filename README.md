# Beyond static snapshots: capturing aging momentum for enhanced lifespan prediction in mice

This repository contains the source code to evaluate and compare the predictive performance of the proposed $F_2^{\mathrm{cdrts}}$ dataset against the $F_1$ dataset, as described in the accompanying paper.

## 1. Environment
The following environment has been tested and verified for operation:

* **OS**: Ubuntu 24.04.2 LTS (on Windows WSL2)
* **Platform**: VS Code (Jupyter Extension)
* **GPU**: NVIDIA TITAN RTX
* **Python**: 3.10.18
* **CUDA (PyTorch)**: 12.6
* **Key Libraries**:
    * `torch`: 2.7.1+cu126
    * `scikit-learn`: 1.6.1
    * `pandas`: 2.2.3
    * `numpy`: 2.0.1
    * `xgboost`: 3.0.5

## 2. Data Availability
The dataset used in this study is publicly available on Mendeley Data.

* **Original Filename**: `Healthspan.data.paired.csv`
* **Source**: [Mendeley Data](https://data.mendeley.com/datasets/kfkhgw359t/1)
* **Setup**: Download the file from the source above, rename it to **`physiological_lifespan_dataset.xlsx`**, and place it in the same directory as the `.ipynb` file.

## 3. Project Structure
* `death_pred_func_mine_update2.py`:
    * Defines regression models used in the study, including Neural Network, Support Vector Regression, and XGBoost.
    * Contains functions for individual-wise strain-stratified cross-validation and data processing.
* `death_prediction_for_comparison2.ipynb`:
    * The main execution notebook for the analysis.
    * Performs data preprocessing and constructs a proposed dataset, $F_2^{\mathrm{cdrts}}$.
    * Executes training and test, calculating Mean Absolute Error and saving the results into **Excel files (.xlsx)**.
    * **Note**: The figures presented in the paper were generated by further processing these exported Excel files.

## 4. How to Run
1. Place `death_pred_func_mine_update2.py`, `death_prediction_for_comparison2.ipynb`, and `physiological_lifespan_dataset.xlsx` in the same directory.
2. Open `death_prediction_for_comparison2.ipynb` in VS Code and execute the cells sequentially.

**Operational Notes:**
* This notebook includes supplementary analysis cells not explicitly featured in the paper. To reproduce the results, please execute the relevant cells corresponding to the manuscript.
* To reproduce the **Ablation Study** (as described in **Section 3.3 of the manuscript**), execute all cells up to `Cell 9`, followed by `Cell 13` through `Cell 15` in order.
